============
Installation
============
1) Go to git project root
2) Open cmd line
3) python setup.py build

==================
Configuration file
==================
It should be created or generated from script in your home directory.
It is named: .copy_test_file_config

========================
Structure of config file
========================
source_path=<path to source directory>
destination_path=<path to destination directory>
file_name=<name of your generated results, for example: emailable-report.html>
